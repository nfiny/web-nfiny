Project Flow

A quick look at the lifecycle of a project, from start to finish.

SCOPING & PLANNING

Gather requirements, collect all the required resources, and scope out any gotcha's. Here, the feasibility of the project will be decided, based on budget, technical requirements, and project delivery date.

DESIGN

We will create wireframes & UX flow, then polished UI views with the input of our clients. This is an iterative and continuous process.

DEVELOPMENT

Development of the project commences. Maintainable and reliable code is the objective, with Unit Tests & security in mind. UAT is performed in this step.

RELEASE & SUPPORT

The project is released, monitored, and tested. Documentation, change requests, and support structures are established.
